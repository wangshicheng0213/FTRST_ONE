Search

Search SNP

In this module, users can search SNP by rsid, gene symbol and select a Minimum P-value of a SNP. For example, if users input some SNPs (one SNP per line) and clik "search" button, they will get the results by rsid. But if users input some genes (also one gene per line), they will get results of those SNPs in input genes. Last, if users also select a minimum P-value, they will get the results with P-value less than they select.

Figure 1 Options for SNP search and SNP search result.

Search Gene

In this module, users can search gene by the sourceof gene, gene symbol or Entrez Id. For example, if users select one or more gene sources and click "search" button, they will get gene results with they select gene sources. Users can also search genes by inputing their interested genes and select interested gene sources.

Figure 2 Options for gene search and gene search result.

Analysis & Tools

Co-expression analysis

In this module, users need to do the following 4 steps: 1. Input genes (one gene per line); 2. Select a brain region; 3. Select a developmental stage; 4. Select a minimum P-value.

Figure 3 Options for Co-expression analysis and analysis result.

Expression pattern analysis (BrainCloud)

In this module, users only need to input interested genes (one gene per line).

Figure 5 Options for BrainCloud expression analysis and analysis result.

Protein protein interaction analysis

In this module, users only need to input interested genes (one gene per line).

Figure 6 Options for protein-protein interaction analysis and analysis result.

Association study analysis

In this module, users can select different mode to generate a regional association plot. If users select SNP tab, they should supply a rsid, a flanking size and select a 1000 Genomes population. If users select gene tab, they should supply a gene name, a flanking size and select a 1000 Genomes population. Index SNP and title is optional. If users select region tab, they should select a chromosome, input start position, end position and select a 1000 Genomes population. Index SNP and title is optional.

Figure 7 Options for regional association study analysis and analysis result.

eQTL analysis

In this module, users need to input interested genes (one gene per line) or interested SNP (one SNP per line) and select a Minimum P-value.
